Root
	
- content.php -> file untuk mengatur tampilan yang dipanggil.
	
- FormRegister.php -> Menampilkan form registrasi 	
- index.php -> untuk memanggil semua file-file, mulai dari css sampai file php
	
- InsertProduct.php -> untuk menjalankan semua aksi ecommerce yang ada di frontend.
	
- ShowCartBeside.php -> menampilkan jumlah belanja di samping.
	
- ShowCart.php -> menampilkan keranjang belanja secara detail.
	
- ShowOrder.php -> menampilkan detail pesanan dari belanja yang telah selesai.
	
- ShowProductDetail.php -> menampilkan produk secara detail ketika ada pemanggilan dari sebuah produk.
	
- ShowProduct.php -> menampilkan semua produk.


Folder Css -> menyimpan file css.
	
- outside.css -> file utama css website, untuk frontend dan backend


Folder Dashboard -> menyimpan folder dan file-file untuk halaman admin.
	
- content.php -> file untuk mengatur tampilan yang dipanggil.
	
- index.php -> untuk memanggil semua file-file, mulai dari css sampai file php
	
	
Folder Buku -> Menyimpan file-file untuk manajemen buku di admin.
		
- delete.php -> untuk aksi delete buku.
		
- edit.php -> Menampilkan form edit buku.
		
- edit.proses.image.php -> aksi edit yang digunakan ketika ada gambar buku yang diupload
 		
- edit.proses.php -> aksi edit buku
		
- input.php -> menampilkan form input
		
- input.proses.image.php -> aksi input yang digunakan ketika ada gambar buku yang diupload
		
- input.proses.php -> aksi input
		
- view.php -> menampilkan menu buku

	
Folder Config -> Menyimpan file koneksi serta library untuk upload gambar.
		
- class_paging.php -> untuk membuat paging
		
- fungsi_thumb.php -> class untuk upload gambar
		
- koneksi.php -> untuk menghubungkan ke database

	
Folder img -> Menyimpan gambar-gambar yang diupload.

	
Folder Order -> menyimpan file-file untuk manajemen order
		
- view.php -> menampilkan semua pesanan dan detail pesanan
		
- proses.php -> proses aksi order yang direquest

	
Folder Penerbit -> Menyimpan file-file untuk manajemen penerbit di admin.
		
- delete.php -> proses aksi delete.
		
- edit.php -> form edit penerbit
		
- edit.proses.php -> proses aksi edit data penerbit.
		
- input.php -> form input data penerbit
		
- input.proses.php -> proses aksi input penerbit
		
- view.php -> menampilkan menu penerbit.
